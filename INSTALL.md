## What You Need

- macOS (Raycast is macOS-only)
- Node.js 18+ (auto-installed if you have Homebrew)
- Raycast app ([Download](https://www.raycast.com/))
- OpenAI API Key ([Get one here](https://platform.openai.com/api-keys))
